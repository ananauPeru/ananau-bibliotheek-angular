Step 1 - Locate the emails

The emails can be found at https://app.sendgrid.com/.

You can log in via the credentials that can be found in the Azure vault.

After logging in, the emails can be found on this platform. You can now add, delete or update emails.

Step 2 - Locate the code

De backend code of the automatic emails can be found on the Azure portal.

The code of an email can be modified by clicking on 'edit' in the 'overview' tab.

Under the "schedules" tab, the email schedule can be modified.
