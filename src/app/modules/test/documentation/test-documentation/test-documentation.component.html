Create a new test

Created: 21/05/2024 | Last Update: 21/05/2024

If you want to create a new test, you can click on "Create test" in the menu or on the dashboard.


You will get a form. Fill in all the required fields. By default, the time limit for a test is 60 minutes. To change it, you can click on "Settings" in the top right corner.


In the appearing window, you can change the time limit for a test in minutes.


After creating a test, you will be redirected to a page with the list of tests. Here you can copy the link of the test and send it to a Spanish learner.


The test is created and the student has a link. After the student submits the test, you can see it in the list of submitted tests.


By clicking on "Actions", you will be redirected to the page with details of the submitted test. There will already be a score for all auto-graded questions. Open questions have to be graded manually. For that, on the top right corner, you can find a button "Edit Score".


Here you have the opportunity to check every open question and grade it separately.
